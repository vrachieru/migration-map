Migration Map

A visual representation of migration patterns

Abstract

Considering tweets provided by Twitter users, the application proposed will be able to generate in real-time an interactive map representing the current intention of humans to move from one place to another. In that way we can visualize if there is a massive migration flow.

Main menu

The application consists of four main sections:

  • Live - view migration chatter in real time.
  • Historical - view migration trends over a period of time with added controls to better grasp the information.
  • Statistics - correlated information about migration chatter over a period of time.
  • Tour - user guide presenting how to get around to using the application.

Left Pane

The left pane is home to this application's key visual aspect, the map.

Here you can visualize migration path either in real time or from historical data.
Information can be represented to best suit your needs by customizing the way the map or underlying details are shown.

To list just a couple of things you can tweak to your own liking:

  • map projection type
  • migration projection type
  • playback speed
  • etc

On the right side there's a little details panel that shows the total currently registered/displayed migration intentions as well as the top countries affected by this phenomenon.

Options panel

As stated above, the options panel offers you the means to customize your experience.
This feature is dynamically linked to each section and will display options relevant to a particular section.

For example, in the live view you don't need the ability to change the speed at which you see the information because it's real time.

  • Representation
    • Arc - animated line progressing from the migration source to it's destination
    • Circle - dual ripple effect, red for the source and green for the destination
    • Particle - particle zooming from source to destination
  • Projection
    • Flat - flat map representation of Earth
    • Globe - spinning sphere (globe) representation of Earth
  • Autorotation
    • available only for the globe projection

In adition to the live view, you can see that in the historical view we have two more controls.

The date displayed is the current moment in time for which the data is displayed.
This is represented in UTC in order to not make any confusions between timezones.

The other new feature is that of being able to control the current point in time for the data being displayed via the slider.
The speed with which time passes in your current representation can be controlled as well.
This is visualised by the playback multiplyer (in the picture below '16x') which is controlled by the fast forwatd and backward buttons on the right.
You can also pause and resume playback with the coresponding button.

Details panel

The details panel can be found in the lower right corner of the live and historical views.
Its purpouse is to act as a heads up display (HUD) to the user displaying minimal but significant statisical information.

The total migration displays the number of migration intentions processed by the application at the current point in time.

The second item displays the top cpuntries affected by the migration process.
This includes both source and destination countries, information which can be toggled with the use of the button in the right hand side of the item's header.

Right pane

The right pane is home to the core information that lays at this application's foundation, the tweets.
Each tweet that is represented on the map has its correspondent within the tweet pane.

The display menner is similar to their original form.
A generic avatar to represent the person, their full name and Twitter handle as well as the month and day when the message originated.

In adition to this, under the avatar lie the flags of the origin and destination countries for the migration.
This provides all the needed information at a glance.

Live view

As previously described, the live view shows information in real time.
That is why there is not much room for data manipulation.

You can see in the image below that the information stream is much lower than that of the historical view where everything is put into high gear.

Historical view

The historical view offers the advantage of being able to better manipulate the data visualisation of data.
By customizing the playback options you can get a better idea of what the migration patterns are as opposed to a low throughput stream of live data.

Statistics

The statistics view offers a tiny bit of extra information in relation to what you can usually see in the historical view
Here you can visualize the global trends color coded for departures, arrivals and the median between the two gropued by country.

On the right side pane you can see the complete list of countries from where migrations have originated sorted in descending order.
Viewing this for a specific timeframe you can correlate the massive migrations in the top countries with events happening there. This could give you a different perspective over your surroundings.
